# CRUD API

This is a simple CRUD API's are developed using Node, Express and Mongodb.

The enpoinds implementd are as follow

```
POST /auth/login
name=&email=&password=

POST /auth/signup
name=&email=&password=

GET /auth/signout


POST /user/import
file=

POST /user/createUser
userData


GET /user


GET /user/:id
id=


PUT /user/:id
id=&userData=


DELETE /user/:id
id=

```
